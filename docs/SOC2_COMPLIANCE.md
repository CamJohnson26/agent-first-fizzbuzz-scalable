# SOC2 Compliance

FizzBuzz Scalable is committed to the highest standards of security, availability, and processing integrity.

## Security Principles

Our platform is designed with security as a first-class citizen:

- **Data Encryption**: All data in transit is encrypted using TLS 1.3.
- **Access Control**: We implement strict Role-Based Access Control (RBAC) for all internal systems.
- **Vulnerability Scanning**: Automated security scans are run on every commit.

## Availability

We guarantee a **99.99% uptime SLA** for our Enterprise customers, backed by our globally distributed Edge Orchestration.

## Processing Integrity

Our [Fidelity Guarantees](FIDELITY_GUARANTEES.md) ensure that every computation is accurate and verified, fulfilling the Processing Integrity requirement of SOC2.

## Reporting

Enterprise customers can request our latest SOC2 Type II report through their dedicated account manager.
