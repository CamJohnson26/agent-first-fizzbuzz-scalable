# Feature: Create Agent Skills

## Description
This feature involves creating a set of mandatory agent skills in `.junie/skills/` to automate and standardize common tasks:
1. `scan-issues`: Scan for and create GitHub issues for bugs, quality, and improvements.
2. `cleanup-worktrees`: Manage and cleanup old worktrees or branches.
3. `create-release`: Standardize the release process (documentation, GitHub release, blog post).
4. `marketing-update`: Automated high-impact marketing blog post generation from project records.
