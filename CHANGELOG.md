# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2026-04-17

### Added
- **v86 Virtual Machine Isolation (F099)**: Secure, emulated x86 Linux environment for isolated execution.
- **v86 Emulated Event Queue (F098)**: Lightweight asynchronous event processing running inside the emulated VM.
- **SQLite Persistence (F041)**: Enterprise-grade local data storage for computation history and audit logs.
- **Lean 4 Formal Verification (F042/F080)**: Mission-critical algorithms formally verified for mathematical precision.
- **Marketing Page Hamburger Menu**: Improved mobile accessibility with a responsive navigation menu.
- **Expanded Marketing Features**: Showcasing new capabilities like Rust WASM and SQLite integration.
- **Agent Policy Updates**: New mandatory guidelines for agents to maintain project visibility (README & Marketing).

### Fixed
- Marketing page mobile links accessibility by adding a hamburger menu.
